## SOP Format Standard
Each SOP includes:
1. **Title & Purpose**
2. **Scope** — who this applies to
3. **Frequency** — when to run this SOP
4. **Materials/Tools needed**
5. **Step-by-step instructions** (numbered)
6. **Decision trees** (if/then scenarios)
7. **Quality checkpoints**
8. **Common mistakes to avoid**
9. **Escalation path**
10. **Version + last updated**
