## Template Format
Each template contains:
1. **Sitemap** — pages and their purpose
2. **Copy blocks** — headline, subheadline, body, CTA per section
3. **Lovable.dev prompt** — drop-in prompt to build with AI
4. **SEO metadata** — title tags, meta descriptions per page
5. **Variable list** — all `{{variables}}` to fill with client data
